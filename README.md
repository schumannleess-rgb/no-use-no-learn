# 不用不学 — No Use, No Learn

> 一个反直觉的学习工具：卡片越少，学得越多。

---

## 为什么只有 7 张卡片

1956 年，认知心理学家 George Miller 发现人类工作记忆的容量大约是 **7±2** 个信息块。超过这个数，大脑就开始丢东西。

大多数学习工具让你囤积——收藏 100 篇文章、截图 50 页笔记、建 200 张闪卡。然后呢？再也没打开过。

**不用不学**反过来做：知识库硬性上限 7 张。加第 8 张之前，你必须先学会一张旧的。这个约束逼你只保留**真正重要**的东西。

名字来自一句大白话：**学了不用，等于没学。**

---

## 知识从哪来

每张卡片不是 AI 凭空编的。

你告诉功利教授一个话题 → 它去 **Wikipedia** 查资料 → 把经过社群审核的事实提炼成 3-7 张教材级卡片。每张卡片有具体数据、有因果逻辑、有使用场景——不是百科摘要，是**值得背下来的东西**。

不确定的内容？AI 会标注出来，或直接告诉你"这个我不清楚"。

---

## 怎么学

打开网页，告诉功利教授你想了解什么。几秒钟后，3-7 张知识卡片出现在你面前。

挑几张有用的加进知识库，学会的点 ✓。卡片颜色按主题自动生成——扫一眼就知道是哪类知识。

没了。就这样。
